# 伝統検定3級 2週間合格ドリル

スマホ学習用の単一HTML教材です。

## 使い方

1. `index.html` をブラウザで開く。
2. 「模試」で全問題/A/B/C/D、分野、全部解く・弱点だけ・ランダム10問/20問を選ぶ。
3. 「カード」で暗記カードをめくり、覚えたカードを記録する。
4. 「要点」と「2週間」で直前確認する。

## スマホに入れる方法

- iPhone: `index.html` をAirDrop、iCloud Drive、またはメールで送り、ファイルアプリから開く。
- Android: `index.html` をGoogle Drive、メール、USB転送などで送り、Chromeで開く。
- PCと同じWi-Fiで試す場合: このフォルダで `python3 -m http.server 4173` を起動し、スマホから `http://PCのIPアドレス:4173/index.html` を開く。

学習記録は開いたブラウザのローカル保存領域に残ります。
